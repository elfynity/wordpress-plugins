A slider to show the reviews from your customer. Autoplay, style options, easy to add and use the shortcode anywhere.





== Description ==
A lightweight plugin designed to show your reviews (testimonials) in an elegant slider using a shortcode. You have styles to choose from and a user dashboard where you can easily add or delete reviews. 

* User Dashboard.  
* Styles to choose from.
*	Responsive, mobile-friendly.
*	Add and delete. 
*	Autoplay, navigation arrows, pause on hover.







== Installation ==
1. Install the plugin and click Activate.
2. Go to Settings -> Devgirl Reviews Slider and Create new reviews.
3. Copy the shortcode:  [devgirl-reviews-slider] and paste it anywhere you would like to show your reviews.
4. The slider will auto size to the width of your web page.
5. Click on the X in the top right of any reviews box to delete that review.

= Questions? =

If you would like to use it in your theme file, use this code in your template page: <?php do_shortcode('[devgirl-reviews-slider]'); ?>
